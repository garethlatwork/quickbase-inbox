QuickBase Inbox

Forward emails to QuickBase

This site forwards email messages to a QuickBase table.
The site will create the QuickBase table for you, or you can specify the table and fields to use.
The email messages come from cloudmailin.com.

Request your own copy of this site below, or get the source code for the site here.

Setup and Test

  1. Sign up for an account on cloudmailin.com.

    Your cloudmailin account should forward email messages to this site, like this:

    https://quickbase-inbox.heroku.com/<quickbase_username@email.com>/<quickbase_password>/-/-/-/-/-/-/-

    • <quickbase_username@email.com> - a QuickBase login email address.
    • <quickbase_password> - the QuickBase login password.

    Note: All the /- parts are required.

  2. Send a test email to your cloudmailin email address.

    Your email message must be in plain text format, not HTML format.
    This will create a QuickBase Email Inbox application, with an Emails table containing one record for your test email.

  3. Your QuickBase Inbox is ready!

Options

You can set a few options using the address of the target web site in your cloudmailin account.
The format of the target web site is:

https://quickbase-inbox.heroku.com/username@email.com/password/realm/table/apptoken/from/to/subject/message.
  1. username@email.com - A QuickBase login email address (required).
  2. password - The QuickBase login password (required).
  3. realm - realm.quickbase.com, if you do not access QuickBase on www.quickbase.com.
  4. table - The ID of your QuickBase Emails inbox table.
  5. apptoken - The application token, if any, required to access your Emails inbox table via the QuickBase API.
  6. from - The ID (a number) of the Email type field in your Emails inbox table that will receive the From email address.
  7. to - The ID (a number) of the Text type field in your Emails inbox table that will receive the To email address(es).
  8. subject - The ID (a number) of the Text type field in your Emails inbox table that will receive the Subject of the email.
  9. message - The ID (a number) of the Text - Multi-line type field in your Emails inbox table that will receive the Message text of the email.

The options you do not use must be replaced by a dash ( /- ).
